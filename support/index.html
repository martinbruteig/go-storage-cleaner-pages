What GO Storage Cleaner does

GO Storage Cleaner helps Pokémon GO players clean up storage safely. It generates conservative search strings that you copy and paste into Pokémon GO yourself: first to find and tag Pokémon worth keeping (protect first), then to review transfer candidates (clean second).

What it does not do

  • No login and no accounts
  • No Pokémon GO account access
  • No Niantic server access
  • No automation of gameplay
  • No exact PvP rank claims — search strings are review helpers based on broad in-game IV buckets
  • It never marks anything as safe to delete — you always review manually before transferring

Basic troubleshooting

A search string returned nothing or looked wrong.
Pokémon GO search syntax occasionally changes. Email us the exact string and what you expected — string fixes ship quickly.

The Copy button didn't work.
Long-press the search string text and copy it manually, then email us your app version (shown in Settings).

Something I tagged or favorited showed up as a transfer candidate.
That should never happen — it's the app's core promise. Email us immediately with the string and a screenshot.
